# 민원 노드용 대표 발화 후보 30선 (Query Pool)

**기준 모수:** raw_query_pool_v3 (사회복지·노인장애인·위기대응, 370건)  
**제외 카테고리:** 보건의료(faq01), 인구아동(faq03)  
**배분:** 세그먼트별 균등 6개 × 5세그먼트 = 30개  
**문서 성격:** TR / Retrieval Coherence Validation / 대표 쿼리셋 용도의 독립 질의 Pool  
**출처 산식:** `https://www.129.go.kr/faq/{faq_type}_view.jsp?n={n-value}`

## 선정 기준 (3축)

| 축                | 지표      | 기준                      | 설명                                       |
| ----------------- | --------- | ------------------------- | ------------------------------------------ |
| ① 세그먼트 적합도 | seg_score | ≥1 필수, ≥2 우선          | 해당 세그먼트 키워드 매칭 수 (상한 3)      |
| ② 발화 생동감     | live      | ≥2 우선                   | 구어체·1인칭·감정 신호 포함 여부 (상한 3)  |
| ③ 갈등 잠재력     | conf      | S4는 ≥1 필수, 나머지 참고 | 불이익·압박·재확인 구조 포함 여부 (상한 3) |

> **이상치 처리 원칙:** seg_score=0이더라도 제목의 의미적 구조가 해당 세그먼트 대화 노드에 부합하는 경우 선정 가능. conf=0이더라도 발화가 담은 상황의 불이익 구조가 명확한 경우 의미 기반으로 재분류. 두 경우 모두 근거란에 명시.
